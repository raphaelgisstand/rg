Validating and Debugging Web Pages

Validating and debugging web pages is a tedious process. When creating web pages that will be viewed over the internet, every page must pass through the W3C Markup Validation Service. The W3C Validator informs you to the number of Errors on a web page. The W3C Validator also gives suggestions on how to correct these errors. After hours of trying to debug my web pages, I now know the importance of closing every opened tag. I also learned that all tags and attributes must be in lowercase and attribute values must be inside quotation marks. Debugging my web pages also gave me a better understanding of source coding.
