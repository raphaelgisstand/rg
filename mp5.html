Dreamweaver Web Page Editor

Dreamweaver is a powerful website editor that allows one to create and edit multiple web pages in separate tab windows. Dreamweaver cuts one's job in half by making source coding easier than having to do it by hand. Dreamweaver lets you insert links and tables much faster than using Notepad or Textwrangler. What separates Dreamweaver from other web page editors is that it allows you to see multiple views while editing. You can see either the code page, the design page (how it looks over the web), and the split view which allows you to see both the coding page and the design view simultaneously. The W3C Validation Service can also be accessed through Dreamweaver to check for errors and bugs on your web pages. The table below shows my web pages before and after using Dreamweaver.
